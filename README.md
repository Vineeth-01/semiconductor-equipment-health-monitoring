# semiconductor-equipment-health-monitoring
## Objective
To simulate health monitoring of semiconductor manufacturing equipment by analyzing sensor data such as temperature, pressure, and vibration using Python-based data analytics.

## Project Description
This project focuses on detecting abnormal operating conditions in semiconductor equipment through trend analysis and threshold-based monitoring. Early identification of equipment degradation helps enable predictive maintenance and improve tool reliability.
